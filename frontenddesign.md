
**Role:** Senior Flutter UI/UX Developer
**Style:** Modern Minimalist / Bauhaus
**Theme:** Maroon (#800000)

**Tugas:**
Buatlah kode Flutter untuk halaman [semua halaman] dengan ketentuan desain berikut:

1. **Konfigurasi Warna:**
   - **Utama:** Maroon (`0xFF800000`) digunakan hanya untuk elemen krusial (Tombol Utama, Header Text, Active Icons).
   - **Background:** Putih bersih (`0xFFFFFFFF`).
   - **Aksen/Surface:** Abu-abu sangat muda (`0xFFF9FAFB`) untuk card atau input field.

2. **Tipografi:**
   - Gunakan font plus jakarta sans
   - Headline menggunakan `FontWeight.w800` (Extra Bold) dengan `letterSpacing: -1.0`.
   - Body text menggunakan `Colors.black87` untuk keterbacaan tinggi.

3. **Komponen:**
   - **Border Radius:** Gunakan nilai kecil (`4.0` atau `8.0`) untuk kesan tegas dan profesional.
   - **Shadows:** Hindari drop shadow yang pekat. Gunakan border tipis atau `elevation: 0`.
   - **Spacing:** Gunakan `SizedBox` yang konsisten untuk menciptakan white space yang luas (minimalis).

4. **Output Kode:**
   - Berikan satu file Flutter modular.
   - Gunakan `Material 3`.
   - Pastikan kode bersih, terdokumentasi, dan responsif.
